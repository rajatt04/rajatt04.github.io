## ✨ Features

- **Hero Section** — Animated developer intro with a Kotlin code snippet aesthetic
- **Tech Arsenal** — Visual showcase of core, architecture, UI & tooling skills
- **Featured Projects** — Live-linked cards pulling from GitHub portfolio
- **Freelance Pricing** — Transparent Starter / Professional / Enterprise tiers with monthly & yearly toggle
- **Cyber Hover Game** — A fully playable in-browser dodge game built with vanilla JS *(Desktop recommended)*
- **Experience & Education Timeline** — Clean vertical timeline with downloadable resume
- **Contact Form** — Direct message + WhatsApp & email quick-connect
- **Fully Responsive** — Optimized for mobile, tablet and desktop
- **Dark Material Theme** — Material 3 Expressive color system throughout

## 🍴 How to Fork & Customize

Want to use this as your own portfolio? Here's how:

### 1. Fork the Repository

Click the **Fork** button at the top right of this page, or:
```bash
git clone https://github.com/rajatt04/rajatt04.github.io.git
cd rajatt04.github.io
```

### 2. Personalize Your Info

Open `index.html` and update your name, tagline and social links.

### 3. Update Your Projects

Find the **Featured Projects** section and replace the cards with your own GitHub repos.

### 4. Update Pricing (Optional)

Edit the **Development Packages** section with your own freelance rates.

### 5. Update Contact Info
```html
<a href="mailto:your@email.com">mail</a>
<a href="https://wa.me/YOUR_NUMBER">phone</a>
```

### 6. Deploy on GitHub Pages

- Rename your fork to `YOUR_USERNAME.github.io`
- Go to **Settings → Pages → Source → Deploy from branch (main)**
- Live at `https://YOUR_USERNAME.github.io` 🎉
